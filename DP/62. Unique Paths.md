代码：
```python
class Solution:

	def uniquePaths(self, m: int, n: int) -> int:
		@cache
		def dfs(i:int,j:int):
			if i<0 or j<0:
				return 0
			if i == 0 and j==0:
				return 1
			return dfs(i-1,j) + dfs(i,j-1)
		return dfs(m-1,n-1)
```
心得：
- base case是到达右下角的时候,它的路径总数是从上面来的路径总数和左边来的路径总数dfs(i,j)=dfs(i−1,j)+dfs(i,j−1)
- 递归边界
	- dfs(−1,j)=dfs(i,−1)=0。无法从 (0,0) 到达这些位置。
	- dfs(0,0)=1。起点到它自己有一条路径，即原地不动。
- 时间复杂度O（mn）
