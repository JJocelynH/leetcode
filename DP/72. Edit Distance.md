难度：Medium

标签：动态规划

题目截图：![](assets/72.%20Edit%20Distance/file-20260204223809550.png)
代码:
```python
class Solution:
	def minDistance(self, word1: str, word2: str) -> int:
		m = len(word1)
		n = len(word2)
		@cache
		def dfs(i,j):
			if i<0:
				return j+1
			if j<0:
				return i+1
			if word1[i] == word2[j]:
				return dfs(i-1,j-1)
			return min(dfs(i-1,j),dfs(i,j-1),dfs(i-1,j-1))+1
		return dfs(m-1,n-1)
```

心得：
- i<0的时候返回j+1是因为这时候已经穷尽了word1，所以需要的操作数是word2剩余所有长度，j<0同理
- min(dfs(i-1,j),dfs(i,j-1),dfs(i-1,j-1))+1,这里加一一定不要忘记，因为不管是删除（dfs(i-1,j)），增加（dfs(i,j-1)），还是替换（dfs(i-1,j-1)），都进行了一次操作，之前没+1所以错了很多次
