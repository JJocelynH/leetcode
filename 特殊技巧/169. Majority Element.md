代码：
```python
class Solution:

	def majorityElement(self, nums: List[int]) -> int:
		ans = hp = 0
		for x in nums:
			if hp == 0:
				ans, hp = x, 1
			else:
				hp += 1 if x == ans else -1
		return ans
```
心得：
- 想象是打擂台，第一个上去的人hp是1，后面如果相同的就加hp，如果遇到不同的就减hp，到最后如果hp为0就换人
