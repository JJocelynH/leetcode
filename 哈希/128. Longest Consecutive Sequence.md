```python
class Solution:

	def longestConsecutive(self, nums: List[int]) -> int:
		ans = 0
		st = set(nums)
		for x in st:
			if x in st:
				x-=1
				continue
			y = x+1
			while y in st:
				y += 1
			ans = max(ans,y-x)
		return ans
			
```
心得：
- 结束的时候y-1是连续数列的最后一个数，所以长度是y-x
