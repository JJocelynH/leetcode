```python
def longestPalindrome(self, s: str) -> str:
	n = len(s)
	ans_left = ans_right = 0
	
	#奇数长度的子串，从任意位置的i开始向两边扩展
	for i in range(n):
		r = l = i
		#注意这里结束条件是l<0，l=0的时候还是合法的
		while r < n and l >= 0 and s[l] == s[n]:
			l -=1
			r += 1
		#while循环结束的时候是[l+1:r-1]
		#长度为 (r-1)-(l+1)+1 = r-l-1
		if r-l-1 > ans_right - ans_left:
			ans_right = r
			ans_left = l+1
	
	for i in range(n-1):
		r = i+1
		l = i
		while l >= 0 and r < n and s[r] == s[l]:
			l -= 1
			r += 1
		if r-l-1 > ans_right - ans_left:
			ans_right = r
			ans_left = l+1
	#结果是左闭右开所以前面所有更新答案都是左闭右开	
	return s[ans_left:ans_right]
```

心得：
- 从任意i开始向左右扩展，分奇偶两种情况
