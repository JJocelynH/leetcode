标签：动态规划

题目截图：![](assets/1143.%20Longest%20Common%20Subsequence/file-20260203214445848.png)
代码：
```python
class Solution:
	def longestCommonSubsequence(self, text1: str, text2: str) -> int:
		m = len(text1)
		n = len(text2)
		@cache
		def dfs(i:int,j:int):
			if i < 0 or j < 0:
				return 0 
			if text1[i] == text2[j]:
				return dfs(i-1,j-1)+1
			return max(dfs(i-1,j),dfs(i,j-1))
		return dfs(m-1,n-1)
```

心得：
- subsequence不是子串，所以是一直删除
- 时间复杂度O(m*n)
