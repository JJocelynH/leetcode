代码：
```python
class Solution:
	def findDuplicate(self, nums: List[int]) -> int:
		slow = fast = 0
		while True:
			slow = nums[slow]
			fast = nums[nums[fast]]
			if slow == fast:
				break
		
		head = 0
		while head != slow:
			head = nums[head]
			slow = nums[slow]
			
		return head
```
心得
- 跟环形链表2是一样的，先用快慢指针算出相遇，设环长为b,从起点到环入口为a，从a走c长度到相遇点，fast比slow多走了k圈，可得：
2*(a+c) - (a+c) = k*b
a+c = k*b
a=k*b-c

所以从起点到入环口的距离等于从相遇点继续走到入环口的距离，所以要令head=0然后随便让快或者慢指针走到相遇，此时这个相遇点就是入环口
